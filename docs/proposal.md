# Florida Springs App — Project Proposal

## Project Overview
The Florida Springs App is a portfolio project designed to help users discover and explore Florida’s natural springs. Users can browse springs via a list or map view, view detailed information about each location, and save favorites for later reference.

The project focuses on clean UI, clear documentation, and a realistic full-stack workflow suitable for a professional portfolio.

## Goals
- Demonstrate full-stack planning and documentation
- Practice API design and frontend data consumption
- Build a scalable project that can be extended over time

## Core Features (MVP)
- Browse springs (list + map)
- View spring details (location, activities, amenities, fees)
- Filter or search springs
- Responsive, accessible UI

## Stretch Features
- User favorites
- Reviews/ratings
- Authentication for write actions
